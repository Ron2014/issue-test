# 整数拆分算法

链接：https://www.nowcoder.com/questionTerminal/bfd8234bb5e84be0b493656e390bdebf
来源：牛客网

题目描述

把m个同样的苹果放在n个同样的盘子里，允许有的盘子空着不放，问共有多少种不同的分法？（用K表示）5，1，1和1，5，1 是同一种分法。

数据范围：0<=m<=10，1<=n<=10。

本题含有多组样例输入。

这个问题可以衍生到游戏的匹配算法的组队模式中 =========>>>

例如队伍满人时为5人，但是玩家可通过单人、2人组队、3人组队、4人组队的方式参与匹配规则中。

为提高匹配效率，会将不同人数的请求放入独立的队列中。如

1. 队列1，单人请求。
2. 队列2，2人组队请求。
3. 队列3，3人组队请求。
4. 队列4，4人组队请求。
5. 队列5，5人组队请求。

之后就会用到对于数字 5 的整数拆分结果，分别对队列取出请求，重新分配队伍。称之为分组，teamId（局外组队） -> groupId（局内分组）。

1. 队列5，每个请求自成分组。
2. 队列4 + 队列1 按 1:1 取队首组合。
3. 队列3 + 队列2 按 1:1 或者 队列3 + 队列1 按 1:2 组合。
4. 队列2 + 队列1 按 2:1 或者 1: 3 组合。
5. 队列1 每5个请求组合.

数字 5 的拆分情况共7种

1. 5
2. 4 + 1
3. 3 + 2
4. 3 + 1 + 1
5. 2 + 2 + 1,
6. 2 + 1 + 1 + 1
7. 1 + 1 + 1 + 1 + 1

用动态规划解决。

为了应对以后可能更多人的情况，需要将其抽象成算法 f(n, n)。而且得到的是解（组合序列），而非值（组合种类）。
