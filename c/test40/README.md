## C 语言在Linux控制台彩色输出

显示颜色格式：\033[显示方式;字体色;背景色m......\033[0m]

其中

\033 为8进制表示的控制字符 'ESC'

...... 为会变色的文本

----------------------------------------------------------
| 字体色 | 背景色 | 颜色描述 |
| ------ | ------ | -------- |
| 30     | 40     | 黑色     |
| 31     | 41     | 红色     |
| 32     | 42     | 绿色     |
| 33     | 43     | 黃色     |
| 34     | 44     | 蓝色     |
| 35     | 45     | 紫红色   |
| 36     | 46     | 青蓝色   |
| 37     | 47     | 白色     |
---
| 显示方式 |     效果     |
| -------- | ------------ |
| 0        | 终端默认设置 |
| 1        | 高亮显示     |
| 4        | 使用下划线   |
| 5        | 闪烁         |
| 7        | 反白显示     |
| 8        | 不可见       |
